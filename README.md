# 🛍️ Marketplace API

Este é um projeto de **API REST** desenvolvida com **Spring Boot**, focada na autenticação de usuários utilizando **JWT e Spring Security**.

## 🚀 Tecnologias Utilizadas

- **Java 17**  
- **Spring Boot 3**  
- **Spring Security**  
- **JWT (JSON Web Token)**  
- **JPA / Hibernate**  
- **Banco de Dados PostgreSQL**
